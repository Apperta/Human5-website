Project Brief

We are developing a mobile app for NHS that targets to move towards a service that better meets every person'sâ€™ individual health and wellbeing needs. This is intended to be done by the users through self-rating on a daily basis and setting up daily/weekly personal goals.

This application specifically targets NHS staff.

Self-Assessment

The application allows each user to rate themselves regularly and setup up goals on a daily/weekly basis.

Progress Evaluation

The user is able to see his/her data in a graphical way and thus evaluate the progress over time.

Data Confidentiality

All the users' data will be encrypted to ensure that confidentiality is maintained.

Cross-Platform

Available on multi platforms to benefits mobile users from both iOS and Android.

Notifcation

Users get notification pop up on their screen when they have a goal that needs to be finished in time.

User Profile

Users are allowed to change their personal settings such as personal information and privacy.
