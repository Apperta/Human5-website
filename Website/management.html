Gantt Chart


This Gantt chart indicates how we hope the second term will be organised and sorted out. We should point that like any plan, this might be changed according to the needs of the moment. This gives a structure and framework of what we plan to do, for how long, and in what arrangement. Although, the time units in the chart are weeks, they show how much time we predict and foresee we'll spend on each task proportionally. This lets us know to what extent a specific undertaking is contrasted with different assignments. At the same time, the number of iterations.

Team Collaboration Tools





Trello is a collaboration Kanban style tool that allows projects to be organised into boards. It allows to keep track of the teamâ€™s tasks and their corresponding deadlines. It also allows to assign those tasks to individual members of the team and add task related attachments to the cards. Trello enables users to create boards which serve as a project which can further be divided into lists and cards which represent the various stages of progress, and tasks & ideas respectively. Collectively, it is very flexible and provides an overview and easy access to essential information.


The team made use of Slack as the main communication channel alongside WhatsApp. Slack caters to a large number of robust features which makes a very efficient team messaging app. It allows creation of separate channels for different discussions, uploading and sharing files, integration with 3rd party apps like Skype for video calls, events synchronisation with calendars and many more. All these features helps the team to keep track of all tasks at hand and communicate in a more organised manner.


To share files and documents across the team, we made use of Google Drive. Google Drive allows multiple members of the team to edit, suggest and view documents simultaneously in real-time. The fulfillment of all essential features coupled with the fact that all members had had prior experience with working with Google Drive made it an apt choice.


We are making use of GitHub as a version control and source code management system. Being under the student program for GitHub, GitHub allows the creation of private repositories which are otherwise not available in case of a free account. It allows us to create different branches on git for different team members, who can make the required amends to the code and push the changes to the master branch. GitHub thus allows us to work on the project efficiently.


We are using Balsamiq Mockups as the tool for creating mockups for our mobile app. Balsamiq acts a GUI mockup and website builder application allowing one to make use of in-built widgets and hence arrange them using drag-and-drop. It also allows version control for the mockups and a few 3rd party extensions to make use of.
